# MIS Login Portal

Web-based login interface for the MIS System.
Users scan the QR code displayed here using the MIS Authenticator mobile app.

## Features

- **Dynamic QR Generation**: Fetches secure QR tokens from the Backend API.
- **Real-time Polling**: Checks session status via `api/auth.js` logic.
- **Secure PIN Entry**: Interface for entering the 6-digit PIN generated by the mobile app.
- **Responsive Design**: Works on desktop and tablet screens.

## Structure

- `login_page/`: Contains HTML, CSS, and JS assets.
- `index.html`: Main entry point.

## Usage

1. Ensure the Backend API is running on `http://localhost:8000`.
2. Open `login_page/index.html` in a web browser.
3. The QR code should load automatically. Scan it with the MIS Authenticator app.
